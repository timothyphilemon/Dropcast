TV Installation Guide

Get JukeCast running on your Fire TV in about 10 minutes. You'll need your computer and the TV on the same WiFi network.

Before you start
⚠️ Create your session ID first
Go to your venue dashboard, click Add TV, and create your 6-digit session ID before touching the Fire Stick. You'll need it during setup.
Step-by-step setup
1
Enable Developer Options on your Fire TV
Go to Settings → My Fire TV → About and click the Fire TV name 7 times rapidly until you see "You are now a developer." Then go to My Fire TV → Developer Options and turn on ADB Debugging and Apps from Unknown Sources.
2
Find your Fire TV's IP address
Go to Settings → My Fire TV → About → Network and note the IP address — it'll look like 192.168.1.xxx. You'll need this for every command below.
3
Install ADB on your computer
ADB lets your computer talk to the Fire TV. Pick your OS:
Open Terminal and run these two commands. The first installs Homebrew (skip if you already have it), the second installs ADB:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install android-platform-tools
Open Command Prompt and run:
winget install Google.PlatformTools
Then close and reopen Command Prompt for the changes to take effect.
4
Connect to your Fire TV
In Terminal (Mac) or Command Prompt (Windows), replace TV-IP with your Fire TV's actual IP address:
adb connect TV-IP:5555
For example: adb connect 192.168.1.199:5555

When prompted on the TV screen, select Allow. You should see "connected to TV-IP:5555" in the terminal.
5
Install SmartTube
SmartTube is a free ad-free YouTube player that JukeCast uses to play videos. Run these two commands:
curl -L "https://github.com/yuliskov/SmartTube/releases/download/latest/smarttube_beta.apk" -o ~/Downloads/SmartTubeNext.apk
adb install -r ~/Downloads/SmartTubeNext.apk
⚠️ Required: Open SmartTube on the TV, go to Settings → General and turn on Stop playback after one video (except queue). Without this, SmartTube will keep autoplaying random videos and JukeCast won't be able to return to your broadcast app.
6
Install JukeCast
curl -L "https://jukecast.tv/jukecast.apk" -o ~/Downloads/jukecast.apk
adb install -r ~/Downloads/jukecast.apk
7
Set permissions
These two commands keep JukeCast running passively in the background so it wakes up automatically for every video request:
adb shell cmd package set-home-activity com.barplay.tv/.MainActivity
adb shell appops set com.barplay.tv android:get_usage_stats allow
8
Open JukeCast and enter your session ID
Launch JukeCast on the TV:
adb shell am start -n com.barplay.tv/.MainActivity
On first launch you'll be asked for your 6-digit session ID — enter the one you created in your venue dashboard. This connects the TV to your account permanently. You won't be asked again.
9
You're live
Your dashboard will show a ● Connected status for that TV. Customers can now go to jukecast.tv, enter your session ID, and start casting.
✓
JukeCast runs quietly in the background. When a customer queues a video it wakes up, plays it, then returns to whatever app was running — Pluto TV, Tubi, Atmosphere TV, or whatever you set in your dashboard.
Troubleshooting
TV shows "Not connected" in dashboard
Make sure JukeCast has been opened at least once and the session ID was entered. Also confirm the TV has an active internet connection.
Videos won't play
Make sure SmartTube is installed and the "Stop playback after one video" setting is enabled. Some YouTube videos block third-party playback — try a different video.
Wrong session ID entered on TV
Uninstall and reinstall the JukeCast app — the setup screen will appear again on first launch so you can enter the correct ID.
adb: command not found
ADB wasn't installed correctly. On Mac, make sure Homebrew is installed and re-run the brew command. On Windows, close and reopen Command Prompt after installing.
Connection refused or timed out
Make sure your computer and Fire TV are on the same WiFi network. Double-check ADB Debugging is enabled in Developer Options, and confirm the IP address is correct.
